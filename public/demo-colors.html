ðŸ’¡ Usage Tips

  • Load order matters: Import Bootstrap CSS first, then colors.css
  • All utilities use !important to ensure they override other styles
  • Use .text-bg-* for automatic text color contrast
  • Combine utilities: class="bg-blue-100 text-blue-900 border border-blue-500"
