üí° Usage Tips

  • Load order matters: Import Bootstrap CSS first, then colors.css
  • All utilities use !important to ensure they override other styles
  • Use .text-bg-* for automatic text color contrast - great for badges!
  • Combine utilities: class="bg-blue-100 text-blue-900 border border-blue-500 rounded p-3"
  • For accessibility, maintain at least 4.5:1 contrast ratio for text
  • Lighter shades (100-300) work best for backgrounds
  • Darker shades (700-900) work best for text on light backgrounds
  • See COLORS-GUIDE.md for complete documentation
